# Ethical turnover analysis

## Installation steps

- Open the terminal (`Ctrl + ù` in VS Code)
- Run `.venv/Scripts/activate`
- Run `uv sync`
- Run `uv run python -m ipykernel install --user --name=EthicalTurnoverAnalysis`
- Close and open VS Code again
- When clicking `Run all` at the top of `main.ipynb`, select `Jupyter Kernel → EthicalTurnoverAnalysis`
